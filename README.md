# 📜 סידור **"קול ישראל – תפילת הרבים המאוחדת"**

### חזון לסידור חדש המאחד את הלבבות בתפילה אחת

הקובץ שהעלית מתאר רעיון עמוק: יצירת **סידור תפילה שמנסח את כל הפניות בלשון רבים**, מתוך תפיסה שתפילה היא כוח של כלל ישראל ולא של היחיד בלבד. 

הנה **גרסה מסודרת ומלאה יותר של הסידור**, עם מבנה ברור, שערים, ותוכן שמוכן לפיתוח עתידי (ספר / אתר / אפליקציה).

---

# 🕍 סידור קול ישראל – מבנה מלא

## 🌅 שער א – הכנת הלב לתפילה

הכנה רוחנית לפני כל תפילה.

### תפילות הכנה

* קבלת מצוות **ואהבת לרעך כמוך**
* קבלת עול מלכות שמים עבור כלל ישראל
* תפילה על אחדות ישראל
* תפילה על תיקון העולם

### נוסח פתיחה

**יהי רצון מלפניך ה' אלוהינו ואלוהי אבותינו**
שנעמוד כולנו כאיש אחד בלב אחד,
ונכוון את ליבנו לעבודתך באמת ובאהבה.

---

# 🌄 שער ב – ברכות השחר (לשון רבים)

## הודיה על החיים המשותפים

### נוסח פתיחה

**מודים אנחנו לפניך מלך חי וקיים**
שהחזרת בנו נשמותינו בחמלה
רבה אמונתך.

### ברכות לדוגמה

* ברוך אתה ה' **הנותן לנו בינה** להבין ולהשכיל.
* ברוך אתה ה' **הפוקח את עינינו** לראות אור העולם.
* ברוך אתה ה' **המלביש אותנו ערומים** ברחמיו.
* ברוך אתה ה' **המכין מצעדי אדם** להוליך אותנו בדרכיו.
